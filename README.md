# 🏥 Hospital Emergency Room Analysis Dashboard

# 📌 Project Overview

This project demonstrates an end-to-end healthcare data analysis workflow. The raw Hospital Emergency Room dataset was first cleaned and preprocessed using Python and then analyzed in Microsoft Excel to build an interactive dashboard that provides meaningful business insights.

##📊 Dashboard Preview

![Dashboard](Dashboard\dashboard_E.png)

# 🎯 Project Objectives

- Analyze Emergency Room patient data
- Monitor patient flow and operational performance
- Track key hospital KPIs
- Visualize patient demographics
- Analyze admission status and department referrals
- Build an interactive dashboard for data-driven decision making

# 🔄 Project Workflow

# 📌 Key Insights

- Patient admissions are almost equally split between admitted and non-admitted cases.
- The average patient wait time is approximately 34.5 minutes.
- Patient satisfaction score indicates room for service improvement.
- General Practice received the highest number of referrals.
- Patient distribution is nearly balanced across genders.
